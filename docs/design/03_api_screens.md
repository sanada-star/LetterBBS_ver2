## 2. API詳細仕様

### 2.1 共通仕様

**リクエスト**:
- Content-Type: `application/x-www-form-urlencoded`（POST時）
- セッションクッキー: `letterbbs_sid`

**レスポンス**:
- Content-Type: `application/json; charset=utf-8`
- 成功時: `{ "success": true, ... }`
- エラー時: `{ "success": false, "error": "エラーメッセージ" }`

### 2.2 GET /api.cgi?api=threads

**通知ポーリング用スレッド一覧取得**

パラメータ:
| 名前 | 型 | 必須 | 説明 |
|------|---|------|------|
| since | string | 任意 | ISO 8601形式の日時。この日時以降に更新されたスレッドのみ返す |

レスポンス:
```json
{
    "success": true,
    "server_time": "2026-03-04T15:30:00+09:00",
    "threads": [
        {
            "id": 42,
            "subject": "件名",
            "author": "太郎",
            "post_count": 5,
            "last_author": "花子",
            "updated_at": "2026-03-04T15:25:00+09:00",
            "is_locked": false,
            "has_image": false
        }
    ]
}
```

### 2.3 GET /api.cgi?api=timeline

**タイムライン取得（文通デスク・通知で使用）**

パラメータ:
| 名前 | 型 | 必須 | 説明 |
|------|---|------|------|
| my_name | string | 必須 | 自分の名前 |
| partner_name | string | 必須 | 相手の名前 |

レスポンス:
```json
{
    "success": true,
    "posts": [
        {
            "id": 100,
            "thread_id": 42,
            "seq_no": 0,
            "author": "太郎",
            "subject": "お手紙",
            "body": "こんにちは...",
            "created_at": "2026-03-01T10:00:00+09:00",
            "thread_subject": "お手紙",
            "direction": "sent"
        },
        {
            "id": 105,
            "thread_id": 42,
            "seq_no": 1,
            "author": "花子",
            "body": "お返事です...",
            "created_at": "2026-03-02T14:00:00+09:00",
            "thread_subject": "お手紙",
            "direction": "received"
        }
    ]
}
```

### 2.4 GET /api.cgi?api=desk_list

**下書き一覧取得**

パラメータ: なし（セッションIDから自動識別）

レスポンス:
```json
{
    "success": true,
    "drafts": [
        {
            "id": 1,
            "thread_id": 42,
            "thread_subject": "お手紙",
            "thread_author": "太郎",
            "author": "花子",
            "subject": "",
            "body": "下書き中の本文...",
            "created_at": "2026-03-04T10:00:00+09:00",
            "updated_at": "2026-03-04T10:30:00+09:00"
        }
    ]
}
```

### 2.5 POST /api.cgi?api=desk_save

**下書き保存**

パラメータ:
| 名前 | 型 | 必須 | 説明 |
|------|---|------|------|
| draft_id | integer | 任意 | 更新時に指定。未指定で新規作成 |
| thread_id | integer | 必須 | 返信先スレッドID |
| author | string | 任意 | 投稿者名 |
| subject | string | 任意 | 件名 |
| body | string | 任意 | 本文 |

レスポンス:
```json
{
    "success": true,
    "draft_id": 1
}
```

### 2.6 POST /api.cgi?api=desk_delete

**下書き削除**

パラメータ:
| 名前 | 型 | 必須 | 説明 |
|------|---|------|------|
| draft_id | integer | 必須 | 削除する下書きID |

### 2.7 POST /api.cgi?api=desk_send

**一括送信**

パラメータ:
| 名前 | 型 | 必須 | 説明 |
|------|---|------|------|
| draft_ids | string | 必須 | 送信する下書きIDのカンマ区切り（例: "1,2,3"） |
| password | string | 任意 | 編集/削除用パスワード（全投稿共通） |

レスポンス:
```json
{
    "success": true,
    "posted": 3,
    "results": [
        { "draft_id": 1, "thread_id": 42, "post_id": 150 },
        { "draft_id": 2, "thread_id": 55, "post_id": 151 },
        { "draft_id": 3, "thread_id": 60, "post_id": 152 }
    ]
}
```

## 4. クッキー仕様

| クッキー名 | 用途 | 有効期限 | パス |
|-----------|------|---------|------|
| `letterbbs_sid` | ユーザーセッションID | セッション（ブラウザ終了まで） | CGIディレクトリ |
| `letterbbs_admin` | 管理者セッションID | 設定時間（authtime分） | CGIディレクトリ |
| `letterbbs_name` | 投稿者名の記憶 | 30日 | CGIディレクトリ |
| `letterbbs_email` | メールアドレスの記憶 | 30日 | CGIディレクトリ |
| `letterbbs_pwd` | 投稿パスワードの記憶 | 30日 | CGIディレクトリ |

**クッキーのセキュリティ属性**:
```
Set-Cookie: letterbbs_sid=xxx; Path=/patio; HttpOnly; SameSite=Lax
```
- `HttpOnly`: JavaScriptからのアクセスを禁止
- `SameSite=Lax`: CSRF対策（クロスサイトの POST リクエストでクッキーが送信されない）
- `Secure`: HTTPS環境の場合のみ付与（init.cgiの設定で制御）

## 5. エラー応答

### 5.1 HTML画面のエラー

テンプレート `error.html` を使用。

```html
<!-- var:error_title -->    エラーの種類（例: "入力エラー"）
<!-- var:error_message -->  エラーの詳細メッセージ
<!-- var:back_url -->       戻り先URL
```

### 5.2 APIのエラー

HTTPステータスコードは常に200（CGI環境での互換性確保）。
`success: false` で判定する。

```json
{
    "success": false,
    "error": "セッションが無効です。ページを再読み込みしてください。",
    "error_code": "SESSION_EXPIRED"
}
```

**エラーコード一覧**:
| コード | 説明 |
|--------|------|
| `INVALID_PARAMS` | パラメータ不正 |
| `SESSION_EXPIRED` | セッション切れ |
| `NOT_FOUND` | リソース未発見 |
| `FORBIDDEN` | アクセス権限なし |
| `LOCKED` | スレッドがロック中 |
| `FLOOD` | 連続投稿制限 |
| `CAPTCHA_FAILED` | CAPTCHA不一致 |
| `SERVER_ERROR` | サーバー内部エラー |
